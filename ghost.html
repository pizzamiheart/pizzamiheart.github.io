Project Description & Motivation

I've never seen a job description and immediately found a way to do the job. I'm glad I got to do this for Ghost.

AI (and similar automation) can speed up the first step of almost anything. For Ghost, this can get a new client to more human results, faster. By scraping for trending topics in a client's industry, analyzing and compressing the results with an LLM, and surfacing some relevant first research and ideas, Ghost writers can get started creating quicker.

I think I can help Ghost grow by making more things more efficient and giving time back to creators.
